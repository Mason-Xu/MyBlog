# 函数 

## 函数的定义

```c++
返回类型 函数名称(参数类型 参数){
    函数主体
}
```

### 局部变量和全局变量



## 再谈main函数 
- 主函数只能有一个
```c++
int main(){
    ...
    return 0;
}
```

- 是无参函数,返回类型是int,所以函数主体的最后要返回0.
- 对计算机来说,main函数返回0的意义在于告知系统程序正常终止

## 以数组作为函数参数
- 数组作为参数时,一位数组不用填写长度,二维数组要.
- 实际调用时,只需要填写数组名
- 数组作为参数时,在函数中对数组函数的修改等同于对原数组元素的修改(这与普通的局部变量不同)
