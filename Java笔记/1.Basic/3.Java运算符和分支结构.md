# Java运算符和分支结构
## 1.运算符:
   1. 算术运算符:+-*/%,++,--    
     **++：单独使用时++a=a++：a=a+1      --同理被使用时++a等于a+1；a++等于a**  
   2. 关系运算符:>,<,>=,<=,==,!= 属于boolean型  
   3. 逻辑运算符:&&,||,!         属于boolean型  
   4. 赋值运算符:=,+=,-=,*=,/=,%=  
   5. 字符串连接运算符:+  
   6. **三目运算符:boolean?数1:数2**  


## 2.分支结构:
   1. if:1条路  
   2. if...else:2条路  
   3. if...else if:多条路  
   4. switch...case:多条路  
      优点:效率高、结构清晰  
 	    缺点:整数、相等  
       break:跳出switch
