## find函数  
`<string>.find(<string>)` 第一次出现的index  未找到返回 -1  

`'s'.find('s') = 0 `   
`'s'.find('') = 0 `   
`<string>.find(<string>,<number>)` 从 number未知2开始查找

## 判断是否是回文
 - 使用 string[::-1] 来使string字符串倒写
 - 然后通过 find函数 判断是否相同 ,返回 1 和 0

```python
word = "madam"
is_palindrome = word.find(word[::-1])
print is_palindrome
```
