# 字符串,变量

## 基本运算

`+ - * / ( )`

print 'string' + 9  // 字符串和变量不能一起  
print 'string' * 9 // 输出 9个string  

## 字符串数组

```
name = 'hello'    
name[0] = 'h'    
name[-2] = 'l'
```

获取子字符串   name[1:3]  不包含最后一个 [start : end]  

## find函数  
`<string>.find(<string>)` 第一次出现的index  未找到返回 -1  

`'s'.find('s') = 0 `   
`'s'.find('') = 0 `   
`<string>.find(<string>,<number>)` 从 number未知2开始查找
